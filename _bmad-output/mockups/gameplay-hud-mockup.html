Design Spec Notes

Top Bar

Coin display: Dark pill with golden border. Coin icon: 28px radial gradient circle with 2.5s pulsing glow. Balance: 17px bold gold.

Level pill: Centered, dark semi-transparent, 18px rounded. Move counter 12px muted below.

Settings gear: 46px, purple gradient, 14px radius.

Settings Panel (updated)

No background. Buttons float directly below gear — clean, minimal.

3D buttons: 50px, 14px radius. Bottom shadow (4px) + top highlight (inset 2px white) creates physical depth. Press state: translateY(3px) to "push in".

ON state: Bright vivid green (#44ff66 → #08b825) with green outer glow. High saturation, lively feel.

OFF state: Desaturated gray button. Uses Material Icons with built-in slash: music_off, volume_off, mobile_off. Icon at 40% opacity.

Restart: Green (same as ON toggles) — positive action.

Exit: Vivid saturated red (#ff4444 → #cc0505) with red outer glow.

Icons: Material Icons Round — music_note, volume_up/volume_off, vibration, replay, exit_to_app.

Bottom Action Panel (updated)

Undo: Arrow icon + mini coin icon + cost number. Clear that it costs coins.

Extra Bottle: Bottle outline icon + "+" + mini coin icon + cost. User immediately understands: "add a bottle, costs coins".

Mini coin: 16px version of top-bar coin icon. Same gradient, border, "$" symbol.

Interactions

3D button press: translateY(3px) + reduced shadow = pushed-in feel.

Settings expand: Staggered scale 0→1, EaseOutBack, 0.05s delay.

Toggle tap: Swap green/gray gradient + swap icon (volume_up/volume_off).
